Anatomia de un "Elemento" HTML

Existen dos tipos de etiquetas, las etiquetas que no llevna cierre, y las que si tienen cierre.

Etiquetas con cierre
Son etiquetas que tienen etiqueta de apertura, también conocida como Opening Tag, y etiqueta de cierre.
Por ejemplo: La etiqueta "ancla", < a href=" "> < / a>
Como tienen etiqueta de cierre, este tipo de elementos llevan un contenido por dentro, como también, cualquier cantidad de atributos.
Etiquetas con "cierre automatico / sin cierre"
Son etiquetas que no tienen una etiqueta de cierre, solo etiqueta de apertura u Opening Tag, y por lo tanto, tampoco llevan contenido por dentro.
Pueden llevar cualquier cantidad de atributos dentro de su anatomía.
Por ejemplo: < link rel="stylesheet" href=""/>

A la etiqueta y a todo su contenido, se le llama "Elemento".
A lo que va dentro de la etiqueta, se le llama "atributo", y lo que va dentro del atributo se llama "valor".
