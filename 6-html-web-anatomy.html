HTML:

HyperText Markup Language, por sus siglas, HTML. Significa que es un lenguaje de marcado de texto.

La primera página web era una simple página blanca que tenía hipervínculos para poder conectarte a otras páginas, y eso es lo que básicamente significa HTML.
Es la opción de saltar entre páginas o contenidos a través de hipervínculos que tenemos en HTML, que básicamente viene a ser una etiqueta.

Anatomía de una Página Web

Esta imágen es solo un ejemplo, pero nos ayuda a entender que ciertas posiciones son fijas, como que la etique "< header > < /header >" va en la posición superior, o que la etiqueta "< footer > < /footer >" siempre irá en la posición inferior, como también hay otras etiquetas que nos ayudan a diferenciar la posición en la que nos encontramos.
Pero gran parte de las páginas web tendrán una anatomía muy similiar a la mostrada.

    Primer Contenedor Principal
  • header: Es el contenedor principal, que dentro llevará el logo o nombre de la empresa, una barra de navegación que por dentro tendrá un apartado de búsqueda para podernos mover dentro de la Aplicación/WebApp, ya sea estática o dinámica.
    • Segundo Contenedor Principal:
  • main: Información principal que se puede dividir por secciones.
    Ejemplo: El feed o lista de publicaciones de una red social, división de posts, división de resúmenes de los artículos que podremos encontrar dentro de la página web.
  • sidebar/aside: Contenido secundario de una página, que usualmente se encuentra a los lados del contenido principal (o main).
    • Tercer Contenedor Principal
  • footer: Esto se encuentra al fondo del sitio web.
    Los footers pueden tener diferentes tamaños, hay páginas que tienen footer con gran cantidad de información, como, contacto de redes sociales, dirección de lugar, menciones, etc; como también hay webs que tienen un footer muy pequeño, con unformación bien resumida, como podría ser un simple párrafo.
    - Hay páginas que su footer no está en una posición habitual, como las páginas con scroll infinito, páginas las cuales producen mucha información, de manera infinita, sin poder llegar a un límite.
    Ejemplos: Facebook, TikTok, Twitter, que producen información de manera infinita cuando haces scroll en su página principal, por ende, colocan su sección "< footer >" en alguna parte lateral de su pagina web, que podría estar dentro de la etiqueta "< aside >" o "< sidebar >".


Resumen y datos extra:

    La web se conforma principalmente de tres conceptos:
  • URL => Uniform Resource Locator. El identificador único del sitio en el navegador (por ejemplo: https://platzi.com).
  • HTTP => Protocolo de transferencia de hipertexto. Es el estándar que se utiliza para enviar datos a través de paquetes entre el cliente y el servidor.
  • HTML => es el código que se emplea para estructurar el contenido de tu web, y darle sentido y propósito.
HTML es el código que se utiliza para estructurar el contenido de tu web, y darle sentido y propósito.
HTML son siglas, Hyper Text Markup Language (Lenguaje de Marcado de Hipertexto).
Hyper Text significa que el texto tiene interactividad, conexión con otros documentos.
Markup significa que le pone etiquetas a los elementos. Por eso también se le conoce como un lenguaje de etiquetas.
HTML es un lenguaje interpretado.
HTML es un estándar, así que no importa desde que navegador, que dispositivo se ejecute, es lo mismo en cualquier sitio. Sigue siendo el mismo código.
